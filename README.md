# Read Me Mortal
A *very* MVP game for our project in the Design Course.

For context, [look at images in docs](docs/). We mostly based this project on the teachers diagrams, but removed everything that wasnt needed to implement the use-cases in as much of a MVP way as possible.

## How to run
1. Have make and Docker/Podman installed.
2. `make`
3. Profit

## How to run tests
1. Have make and Docker/Podman installed.
2. `make test`
3. ProfitS
